== Installation ==

It's really easy and super simple to install **Elementor Integrations** plugin but before installing **Elementor Integrations** make sure you are really an [Elementor & ElementorPro](https://wordpress.org/plugins/elementor/) user.

= Automatic Installation =
1.  Go to `Plugins > Add New` screen in WordPress.
2.  Search for **Elementor Integrations**.
3.  Install and activate the plugin, that's it.

= Manual Installation =
1.  Download [Elementor Integrations](https://downloads.wordpress.org/plugin/integrations-for-elementor.zip "Download Elementor Integrations").
2.  Extract the `integrations-for-elementor.zip` file. You'll get plugin files inside `integrations-for-elementor.zip` directory.
3.  Upload the plugin files to the `/wp-content/plugins/integrations-for-elementor` directory.
4.  Activate the plugin through the 'Plugins' screen in WordPress.
